# Название: Игровая библиотека "Геймбук"
## библиотека для управления коллекцией игр, отслеживания прогресса, достижений и сортировки по жанрам (мб получиться реализовать список рекомендованных игр).
## Функционал:   
* Поиск игр по жанру, названию   
* Авторизация через steam   
* Активность в играх за последние 2 недели
* Отслеживать общее число часов проведенных  в играх
* Система отслеживания редкости полученных достижений

## Интерфейс:
* В верхней части страницы находится кнопка «Авторизация через Steam». После входа она заменяется на мини-профиль: аватарка пользователя, никнейм
* **Страница мини-профиля:** находится основной блок со списком всех купленных игр. Изначально он загружается из базы данных. Каждая строка содержит маленькую иконку игры, название и общее время в игре. Там же игры можно будет фильтровать
* **Страница игры:** Описание игры, жанры и дата последнего запуска, Список всех ачивок, подтянутых через API. Возле каждой ачивки стоит чекбокс
* **Страница инвентаря:** Выпадающий список с выбором игры, так как инвентарь для каждой игры запрашивается отдельно. Интерактивная таблица с изображениями скинов/карточек. При клике на предмет открывается боковая панель с его описанием, редкостью (может добавлю открытие ссылки  для этого предмета на ТП)
